Bio Protech Account CRM - 사용자 매뉴얼

Pilot 테스트 버전 | 2026년 4월 접속 URL: https://bioprotech-account-crm.web.app

1. 시스템 개요

Account CRM은 기존 거래 고객의 관계 유지, 재구매, 업셀링을 체계적으로 관리하기 위한 시스템입니다.

기존 Pipeline CRM(신규 딜 관리)과 동일한 Firebase 프로젝트를 사용하며, 모든 데이터는 실시간으로 동기화됩니다.

핵심 목표

  • 고객별 인텔리전스 정보를 체계적으로 축적
  • 사업계획 대비 수주 Gap을 원인별로 분석하고 대응 방안 수립
  • 주니어 AM들이 고객에게 무엇을 물어야 하는지 가이드 제공
  • 월간 리포트에서 숫자뿐 아니라 왜 Gap이 발생했는지 심층 분석

2. 로그인 및 화면 구성

2.1 로그인

  • 접속 시 사용자 선택 화면이 표시됩니다
  • 본인 이름을 선택하면 해당 권한으로 접속됩니다
  • Haksu로 로그인 시 관리자 권한 (전체 고객 편집 가능)

2.2 메인 화면 구성

영역 설명
좌측 사이드바 메뉴 내비게이션 + 요약 정보
상단 탭 대시보드 / 고객 목록 / 리포트 / 진도관리
본문 선택한 메뉴의 콘텐츠

3. 대시보드

대시보드는 전체 현황을 한눈에 파악할 수 있는 홈 화면입니다.

표시 정보

  • KPI 카드: 전체 고객수, 평균 Intelligence Score, 이번 달 활동 건수, Open 이슈, YTD 달성률
  • 긴급 알람: Score 50% 미만 + 미접촉 30일 초과 고객
  • 알람 목록: 재구매 예상일 경과, 계약 만료 임박, 유형별 관리 알람 등
  • Open 이슈 현황: 미완료 활동 로그
  • 긴급 관리 대상: Score 낮은 고객 목록
  • 고객유형별 관리 현황: 유형별 체크리스트 진행률

5. 고객 상세 카드 (모달)

고객을 클릭하면 9개 탭으로 구성된 상세 카드가 열립니다.

5.1 기본정보 탭

고객의 기본 정보를 관리합니다.

  • 회사명, 국가, 지역
  • 사업형태: 변경 시 이력이 자동 기록됩니다 (예: Single → Multiple = 재구매 전환 성공)
  • 담당자, 계약 상태, 거래 시작일
  • 담당 제품군 (복수 선택)
  • Key Contact: 이름, 직책, 이메일, 전화, 결정권자(DM) 여부

5.2 Score (Intelligence Score) 탭

6개 카테고리 × 총 28개 항목을 체크하여 고객 정보 파악 수준을 측정합니다.

카테고리 가중치 파악해야 할 내용
기본정보 완성도 10% 설립연도, 직원수, 매출규모, Key Contact, DM, 조직구조
시장·채널 이해도 15% 시장 규모, 성장률, 유통채널, 수요 부서, 연간 소비량
경쟁사 파악 20% 경쟁 제품, 경쟁사 가격, 자사 점유율, 강약점, 전환 가능성
Pain & Needs 25% Pain Point, 미충족 니즈, 구매 장벽, 기회 요인, 우선순위
거래조건 파악 20% 적용 단가, 결제조건, 계약 만료일, MOQ, 할인 구조
관계 깊이 10% DM 접근, 최근 방문, 신뢰도, 다채널 관계

사용법: 각 항목의 입력란에 정보를 기입하면 자동으로 체크됩니다. Score가 높을수록 해당 고객에 대한 이해가 깊다는 의미입니다.

5.3 Activity (활동 로그) 탭

고객과의 모든 활동을 기록합니다.

이슈 유형: 일반컨택 / 가격협의 / 품질클레임 / 입찰 / 샘플요청 / 계약갱신 / 규제·인증 / 시장정보수집 / 수주활동 / 영업미팅 / 크로스셀링 / VOC수집

  • '+ 새 로그' 클릭 → 유형 선택 + 내용 작성 + 다음 액션 + 기한
  • 수주활동: 세부 유형(견적발송/오더접수/PO접수 등), 예상 수주금액 입력 가능
  • 리스크 경고: 고객 유형에 따라 부적절한 활동 시 경고 메시지 표시
    • 예: OEM 고객에게 '가격협의' 선택 시 → "가격 중심 대화를 지양해야 합니다"
  • 상태 관리: Open → 진행 중 → 완료

5.4 수주이력 탭

확정된 수주 건을 관리합니다.

  • 오더일, 수주금액, 제품군, 오더번호 등 기록
  • 월별/분기별 트렌드 확인

5.5 GAP분석 탭 (핵심 신규 기능)

사업계획 대비 수주 Gap을 분석하고 대응 방안을 수립합니다.

YTD 수주 Gap 현황 (자동 계산)

  • 사업계획과 수주이력을 자동 매칭하여 연간목표/YTD목표/실적/Gap/달성률 표시

Gap 원인 분석

수주 미달의 원인을 태깅합니다 (복수 선택):

원인 설명 AM이 확인해야 할 사항
수요 감소 고객 소비량 감소 고객의 연간 예산 변동, 재고 수준 확인
경쟁 이탈 경쟁사 진입/가격공세 경쟁사 제품/가격 파악, 전환 이유 확인
가격 장벽 단가 불만, 예산 제약 가격 기대치, 볼륨 디스카운트 가능성 협의
채널 문제 유통구조 변화, 재고 적체 재고 현황, 리오더 시점, 채널 변경 여부
인증/규제 인증 만료, 규제 변경 인증 상태, 필요 서류, 예상 소요 기간
관계 약화 담당자 교체, 미접촉 신규 담당자 파악, 미팅 일정 수립
시점 차이 발주 지연, 예산 이월 예산 사이클, 다음 발주 예상 시기 확인
내부 이슈 납기/품질 문제 이슈 해결 상태, 고객 피드백 확인

Tip: '상세 설명'란에 구체적 상황을 기록하면 월간 리포트에서 분석 시 유용합니다.

고객 예산/구매 사이클

  • 예산 편성 시기: 고객의 연간 예산이 결정되는 시점
  • 구매 의사결정 주기: 월간/분기/반기/연간/수시
  • 경쟁 동향 메모: 최근 경쟁사 움직임 기록

기회 파이프라인

Gap을 만회할 수 있는 기회를 등록합니다:

  • 유형: 업셀링 / 크로스셀링 / 신규입찰 / 계약갱신 / 가격구조개선 / 신규채널
  • 대상 품목: 해당 기회의 타겟 제품
  • 예상 금액 + 수주 확률: 가중합계가 자동 계산됩니다
  • 예상 수주 시기: 언제 수주가 예상되는지

이번 달 액션 플랜

고객별로 이번 달 실행할 3~5가지 액션을 등록합니다.

  • 각 액션은 체크박스로 완료 표시
  • 월간 리포트에서 실행률이 집계됩니다

5.6 가격·계약 탭

제품별 단가 및 계약 조건을 관리합니다.

5.7 FCST (Forecast·트렌드) 탭

분기별 수주 예측과 실적 트렌드를 관리합니다.

5.8 크로스셀링 탭

기존 고객에게 추가 제품을 제안하는 기회를 관리합니다.

  • 상태: 미접촉 → 제안중 → 샘플진행 → 수주완료

5.9 유형가이드 탭

고객 유형(OEM/Private/Multiple 등)에 따른 비즈니스 프로세스 가이드입니다.

  • 추천 다음 활동: 체크리스트 미완료 항목 기반 우선 행동 제안
  • 고객 정의, 특성, 핵심 목표, 전략, 리스크 행동
  • 운영 프로세스: Step별 진행 흐름
  • 체크리스트: 유형별 필수 관리 항목

6. 종합 리포트

6.1 주간 리포트

  • 금주 활동 요약 (담당자별 컨택 건수)
  • 사업계획 YTD 진도
  • 분류별 실적 (담당자/품목/지역/사업구분/고객유형)
  • Open 이슈 & 기한 초과 이슈

6.2 월간 리포트

기본 실적 (Section 1~5)

6.3 Excel 다운로드

우측 상단 'Excel 다운로드' 버튼으로 주간/월간 리포트를 Excel 파일로 저장할 수 있습니다.

7. 진도관리

사업계획 대비 월별 수주 진도를 관리합니다.

  • 담당자별/고객별 월별 목표 대비 실적
  • 수주율 게이지 바로 시각적 확인

8. 유형가이드 (별도 페이지)

사이드바 메뉴의 📖 유형가이드를 클릭하면 고객 유형별 비즈니스 프로세스 가이드를 별도 페이지에서 확인할 수 있습니다.

고객 상세카드의 유형가이드 탭은 개별 고객에 대한 체크리스트를 관리하는 곳이고, 이 별도 페이지는 모든 유형의 전략을 한눈에 비교하고 학습하는 참고 자료입니다.

8.1 유형별 핵심 비교표

6가지 고객 유형의 핵심 목표, 전략 요약, 주의사항을 한 테이블에서 비교합니다.

유형 핵심 목표 전략 핵심 주의사항
OEM 장기 공급계약, 단가인하 최소화 전략적 파트너십(KAM), 연간 단가 협상 가격 중심 대화 지양, 구두 합의 금지
Private Label 대리점 전환, 구매량 확대 독점/비독점 계약, 공동 마케팅 브랜드 전략 개입 주의
Multi Order 반복 주문 유지, SKU 확대 주문패턴 분석, 리오더 제안 수동적 대응 금지
Single Order 재구매 전환, 고객 경험 개선 사후관리(7/30/90일), 업셀링 제안 F-up 방치 금지
입찰 낙찰률 향상, 로컬판매 전환 입찰 데이터 분석, 경쟁사 DB 미검토 참여 금지
가격민감 이익 유지, SKU 확대 패키지 제안, 가격 외 차별화 일괄 가격조정 금지

8.2 유형 상세 보기

각 유형 카드를 클릭하면 상세 정보가 펼쳐집니다:

  • 고객 정의: 이 유형이 무엇인지
  • 고객 특성: 이 유형 고객의 행동 패턴
  • 핵심 목표: 이 유형에서 달성해야 할 목표
  • 영업 전략: 구체적인 전략 항목
  • 주의사항: 절대 하면 안 되는 리스크 행동
  • 운영 프로세스: Step별 진행 흐름
  • 관리 체크리스트: 유형별 필수 관리 5항목

카테고리 (가중치) 핵심 질문 예시

경쟁사 파악 (20%) 어떤 경쟁 제품을 사용 중? 경쟁사 단가는? 전환 가능성은?
거래조건 파악 (20%) 현재 적용 단가는? 결제조건은? 계약 만료일은? MOQ는?
시장·채널 이해도 (15%) 시장 규모는? 유통 채널 구조는? 연간 소비량은?
기본정보 완성도 (10%) 직원수/매출규모는? DM은 누구? 조직 구조는?
관계 깊이 (10%) DM 직접 연락 가능? 최근 방문/미팅은? 정보 공유 수준은?

활용법: 미팅 전 고객의 Score 탭을 확인하여 미비한 카테고리를 파악하고, 이 질문 가이드를 참고하여 미팅에서 해당 정보를 수집하세요.

9.4 사업형태 동기화

사업계획의 고객 유형 정보를 고객 계정에 일괄 반영합니다.

10. AM을 위한 실전 가이드

10.1 매일 해야 할 것

  1. 대시보드 확인: 긴급 알람과 Open 이슈 확인
  2. Activity 기록: 고객과의 모든 컨택을 로그에 기록
  3. 액션 완료 체크: 이번 달 액션 플랜 진행 상황 업데이트

10.2 매주 해야 할 것

  1. Intelligence Score 업데이트: 새로 파악한 고객 정보 입력
  2. 주간 리포트 확인: 금주 수주/활동 실적 점검

10.3 매월 해야 할 것

  1. GAP분석 탭 업데이트:
    • Gap 원인 재평가 (상황 변화 반영)
    • 기회 파이프라인 업데이트 (새 기회 추가, 확률 조정)
    • 다음 달 액션 플랜 수립
    • 경쟁 동향 메모 갱신
  2. 유형가이드 체크리스트: 진행 상황 업데이트
  3. 월간 리포트 리뷰: 심층 Gap 분석 결과 확인

10.4 고객 미팅 전 체크리스트

미팅 전에 해당 고객의 상세 카드를 열어 확인하세요:

  1. Score 탭: 어떤 정보가 부족한지 확인 → 미팅에서 파악
  2. GAP분석 탭: 이 고객의 Gap 원인은? → 해결 방안 논의
  3. 유형가이드 탭: 이 유형 고객에게 어떤 접근이 필요한지 확인
  4. Activity 탭: 지난 컨택 내용 복기 → 후속 조치 확인

11. 자주 묻는 질문

Q: 데이터가 저장되지 않아요 A: 화면 하단의 'Firestore 연결됨' 표시를 확인하세요. 인터넷 연결이 필요합니다.

Q: 다른 담당자의 고객을 수정할 수 없어요 A: 본인 담당 고객만 편집 가능합니다. 관리자 권한이 필요하면 Haksu에게 문의하세요.

Q: Intelligence Score가 0%인데 어떻게 올리나요? A: Score 탭에서 각 항목의 입력란에 정보를 기입하면 자동으로 점수가 올라갑니다. Pain & Needs(25%)와 경쟁사 파악(20%) 카테고리가 가중치가 높으니 우선 채우세요.

Q: GAP분석에서 YTD Gap이 표시되지 않아요 A: 사업계획 데이터가 업로드되어 있어야 합니다. 설정 → 사업계획 업로드에서 확인하세요.

Q: Pipeline CRM과 데이터가 연결되나요? A: 같은 Firebase 프로젝트를 사용하지만 컬렉션이 분리되어 있습니다. 사업계획 데이터는 공유됩니다.

용어 설명
Intelligence Score 고객에 대한 정보 파악 수준 (0~100%)
